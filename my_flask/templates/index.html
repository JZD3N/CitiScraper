CitiNewsOnline Web Scraper

This is a web scraper that allows you to search Citi News Room articles by keyword. It will download a CSV file containing the search results. The CSV file will contain the article title, link, date and excerpt.

To use the web scraper, simply enter your search query or incase you have multiple query parameters you can chain them using '+' eg.police+crime.

You can set the value as 0 if you want to scrape all articles regarding your keywords but it will take some time.

To convert the CSV file into an Excel file in Microsoft Excel, please follow the steps below:

  1. Open a new Excel sheet
  2. Click on the Data tab
  3. Click on From Text
  4. Select the CSV file downloaded from the web scraper
  5. Click Import
  6. Select Delimited Text
  7. Select Comma as the delimiter
  8. Click Finish
